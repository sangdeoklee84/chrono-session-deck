# 크로노스피어 에셋 · 스테이지 제작 세션 — 발표 덱

게임 **크로노스피어(Chronosphere)**의 AI 에셋·스테이지 제작 과정을 5단계 슬라이드로 정리한 발표 자료입니다.

## 5단계
1. **진단 & 화풍 정의** — 한 게임, 두 용도(마을 GUI / 스테이지 배경), 스타일 가이드 2종
2. **생성 파이프라인** — Higgsfield MCP(`gpt_image_2`) · Gemini/GPT 스크립트 · 웹 수동 · i2i 스타일 앵커
3. **마을 GUI 에셋** — 셀셰이딩 + 베이스 타일 footprint + 색 정교화 + 투명 컷아웃
4. **인게임 스테이지 배경** — 페인터리 · 2048×1024 · 플레이필드/HUD 세이프존 · 가로 루프
5. **팀 배포 & 온보딩** — 용도별 repo 분리 + 공개 온보딩 페이지
